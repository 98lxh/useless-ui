# 图标 Icon

## 基本用法

通过设置`name`属性加载对应的`icon`

:::demo 

```vue
<template>
  <use-space>
    <use-icon name="user" />
    <use-icon name="snow" />
    <use-icon name="success" />
    <use-icon name="eye" />
    <use-icon name='calendar' />
  </use-space>
</template>
```
:::
